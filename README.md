# meditator

Quick and dirty ambient audio app for meditation and mindfulness.

- TypeScript
- Browser modules
- CSS nesting
- Web Audio API
- Manifest for standalone use

---

## Development

- `npm start` to run a local `lite-server`.
- `npm watch` for TypeScript compilation via `tsc`.
